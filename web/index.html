Privacy and Control

Everything stored in your Space is fully encrypted. No one can access your data or open your encrypted files. With your personal private key, you can control privacy settings and create permissions around who can access your files.

Security

Data is secured using decentralized cloud storage powered by the Sia blockchain. Sia is the only fully decentralized data storage platform without middlemen or a single point of failure. To learn more about Sia, visit: Sia website

Technology

SkySpaces runs completely on your browser and connects to the Skynet CDN for accessing your data on Sia Storage. You may choose to use the Skynet free public portal, a paid portal, or your own portal locally connected to the Sia blockchain. To learn more about Skynet, visit: Skynet Portal

Open-source

The SkySpace code is open-source. This means that anyone can review the code and verify it using Git commit, as well as run their own versions of SkySpace.
